终端隧道：用 tty-share 随处分享你的终端

Section titled “终端隧道：用 tty-share 随处分享你的终端”

是否曾经需要与同事共享终端会话，进行结对编程、远程调试或协作排查问题？tty-share 是一款极其简洁的工具，让你能够在 Internet 上共享 Linux 或 macOS 终端，而远程参与者无需任何设置。

功能介绍与核心亮点

Section titled “功能介绍与核心亮点”

tty-share 是一款轻量级、开源的终端共享解决方案，以其简洁性脱颖而出：

为什么选择 tty-share？

Section titled “为什么选择 tty-share？”
  • 零远程设置：加入的人无需安装任何工具——可直接从浏览器加入
  • 单一二进制文件：用 Go 编写，无依赖——可在 Linux、macOS 甚至 Raspberry Pi 上运行
  • 完全开源：代码库完全透明，可在 GitHub 上查看
  • 安全连接：机器与中继服务器之间的 TLS 加密通信
  • 灵活共享：可通过公共 Internet 或仅限本地网络共享
  • 只读模式：可选择创建只读会话用于演示

工作原理

Section titled “工作原理”

当你通过 Internet（NAT 外部）共享终端时，tty-share 会连接到代理服务器，该服务器协调参与者之间的通信。官方服务器运行在 tty-share.com，但你也可以托管自己的服务器以获得完全控制权。
