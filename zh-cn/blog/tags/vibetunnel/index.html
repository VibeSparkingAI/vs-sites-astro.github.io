VibeTunnel：一键把浏览器变成 Mac 终端，随时随地 Vibe Coding

2025年7月18日
Vibe Sparking

在一台 Mac 上安装 VibeTunnel 后，你可以直接在PC、手机和Pad浏览器里打开 查看、输入、甚至同时开启多路终端，而无需本地安装 SSH、端口映射或其他繁琐配置——真正做到“电脑在家跑，手机也能敲”(Vibetunnel)。

为什么需要 VibeTunnel？

Section titled “为什么需要 VibeTunnel？”
  • 远程盯盘 & 运维：出门在外也能随时查看 build 日志或监控 AI Agent 状态，不用再连 VPN 或开 VNC(GitHub)。
  • 团队协作：通过浏览器把终端共享给同事，省去共用账户或复杂 SSH 隧道的安全隐患(GitHub)。
  • Vibe Coding：Mac、iPad、甚至一部旧安卓机都能即时「接管」本地 shell，真正随走随编(Vibetunnel)。

核心亮点

Section titled “核心亮点”
功能说明
🖥 浏览器即终端基于 xterm.js + Node 后端，实时渲染输出、支持 ANSI 颜色与滚动缓冲区，无需 SSH(GitHub)
⚡ 零配置安装后点击菜单栏图标即启动，本地默认 localhost 只允许自己访问，安全又省事(GitHub)
🤖 AI‑Agent 友好动态标题会显示 Claude/Gemini 等命令的「Thinking / Crafting」状态，方便远程观察长任务(GitHub)
🔒 多重认证本地帐号、SSH Key、环境变量密码或 Token 等五种认证模式，亦可配合 Tailscale/ngrok 暴露到公网(GitHub)
📱 跨端访问Web 界面自适应移动端；官方 iOS 客户端与 Tauri 桌面端也在路上(GitHub)
🎥 会话录制自动以 asciinema 格式保存，方便回放或生成 DemoGIF(GitHub)

快速上手

Section titled “快速上手”

1⃣ 安装

Section titled “1⃣ 安装”

  • macOS 原生 App（推荐）

    仅支持 Apple Silicon，安装时会把 vt 命令软链到 /usr/local/bin(GitHub)。

  • npm 全平台包 服务器、Docker 或 Intel Mac 可用：npm i -g vibetunnel，需 Node 20+(GitHub)。

2⃣ 启动

Section titled “2⃣ 启动”

vt 实际调用 vibetunnel fwd，还能解析你 .zshrc / .bashrc 中的 alias，让 gs=git status 等别名直接可用(GitHub)。

3⃣ 远程访问三板斧

Section titled “3⃣ 远程访问三板斧”
  1. Tailscale：最推荐，创建端到端加密的私有 VPN，只要记住 my-mac.tailnet.ts.net:4020 即可随时连入(GitHub, GitHub)。
  2. ngrok：一键获得 HTTPS 公网地址，适合临时分享给外部伙伴(GitHub)。
  3. 局域网 / Cloudflare Tunnel：在可信网络内直接开放 0.0.0.0:4020，或用 cloudflared 生成临时域名(GitHub)。

场景示例

Section titled “场景示例”
  • 🚄 高铁上用 iPad 查看 CI 日志，任务完成即远程触发下一步构建。
  • 🧑‍💻 远程 Pair Coding，让同事直接在浏览器里帮你定位线上 Bug。
  • 📟 旧手机秒变「仪表盘」，实时盯着 Claude 或 Gemini 的输出流。

最新进展

Section titled “最新进展”
  • UI 大修：Beta 11 起引入更现代的配色与可折叠侧边栏，手机上也能全屏敲命令(GitHub)。
  • 性能与稳定性：Beta 12 解决了 SSH Agent、Linux 安装及欢迎页卡顿等问题，体验更丝滑(GitHub)。
  • 项目活跃度：短短一个月已破 2.8 k commits、14 万行代码，迭代速度令人咋舌(steipete.me)。
  • 幕后花絮：它竟是在 24 小时黑客松里，用 Claude Code + 咖啡一把梭哈出来的！(steipete.me)

使用小贴士

Section titled “使用小贴士”
  1. 安全第一：在公网暴露时务必开启密码或 SSH Key，并限制只读 Session；Dev 环境可以 --no-auth 但切勿生产使用(GitHub)。
  2. Title Hack：在脚本里加 vt title "Deploy 🚀"，浏览器标签栏一眼就能看出哪条流水线在跑(GitHub)。
  3. 配合 Tailscale Funnel：需要给客户临时演示？用 Funnel 把 localhost:4020 映射成一次性 HTTPS 链接，演示完即关(Tailscale)。

结语

Section titled “结语”

如果你想要一个“随时随地，掏出手机就能操作 Mac 终端”的方案，VibeTunnel 基本覆盖了从快速安装、远程安全到 AI‑Agent 监控的全部需求。试着在周末装一把，然后拍张“我在海边也在 vim”的照片，朋友圈立刻多 10 个点赞😉。
