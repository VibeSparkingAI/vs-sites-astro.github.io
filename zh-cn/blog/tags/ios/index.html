Chat UI Swift：展示 AnyLanguageModel 和 Swift-HuggingFace 的 SwiftUI 聊天应用

Section titled “Chat UI Swift：展示 AnyLanguageModel 和 Swift-HuggingFace 的 SwiftUI 聊天应用”

在 Apple 平台上构建 AI 驱动的聊天应用程序，传统上需要面对多个 SDK、API 和认证流程的复杂性。mattt/chat-ui-swift 通过一个精美的 SwiftUI 参考实现简化了这一局面，展示了如何通过统一的 API 集成多个语言模型提供商。

什么是 Chat UI Swift？

Section titled “什么是 Chat UI Swift？”

Chat UI Swift 是由传奇开发者 Mattt Thompson 创建的 SwiftUI 聊天应用，他也是著名的 AFNetworking 和 NSHipster 的作者。这个应用作为两个强大的 Swift 库的演示项目：

  1. AnyLanguageModel - Apple Foundation Models 框架的替代方案，支持自定义语言模型提供商
  2. swift-huggingface - Hugging Face API 的 Swift 客户端

GitHub 仓库： mattt/chat-ui-swift 作者： Mattt Thompson 许可证： MIT

核心特性

Section titled “核心特性”

已支持的功能

Section titled “已支持的功能”
  • Apple Intelligence - 与 Apple Foundation Models 的原生集成（macOS 26+）实现本地 AI
  • Hugging Face 集成 - 通过 OAuth 2.0 认证连接 Hugging Face
  • 流式响应 - AI 响应的实时流式传输，提供流畅的用户体验
  • 聊天持久化 - 保存和管理多个对话

计划中的功能

Section titled “计划中的功能”
  • MLX 模型支持 - 使用 MLX 本地下载和运行模型
  • CoreML 集成 - 支持 CoreML 优化的模型
  • GGUF 格式支持 - 从 Hugging Face Hub 加载 GGUF 模型
  • 模型下载 - 直接从 Hugging Face 浏览和下载模型
  • BYOK（自带密钥） - 支持 OpenAI、Anthropic 等推理提供商

Section titled “理解 AnyLanguageModel”

chat-ui-swift 背后的真正力量是 AnyLanguageModel，这是一个提供统一 API 访问多个语言模型提供商的 Swift 包。
