Section titled “引言：从视频帧到手绘故事”

RanFeng 开发的 ClipSketch AI 提供了一个创新的解决方案。这款开源 Web 应用利用 Google Gemini 的多模态 AI，自动将视频瞬间转化为精美的手绘风格故事板插画，并配有 AI 生成的社交媒体文案。

Section titled “什么是 ClipSketch AI？”

ClipSketch AI 是一款专为视频创作者、社交媒体运营者和二创爱好者设计的 AI 驱动内容创作工作台。它超越了简单的视频播放功能，集成了帧级标记功能和 Google Gemini 最新的多模态大模型，可生成手绘风格的故事板和爆款社交媒体文案。

引言：为什么要为隐私泄露付费？

Section titled “引言：为什么要为隐私泄露付费？”

云端听写服务已成为常态，但它们带来了显著的代价：每月订阅费（10-15 美元）、完全依赖互联网连接，以及令人不安的现实——你的语音数据在远程服务器上被处理。

VoiceFlow 由 infiniV 开发，提供了一个引人注目的替代方案。这款开源 Windows 应用将 OpenAI 强大的 Whisper 语音识别模型直接带到你的本地计算机。你说的每一个字都在你的硬件上完全处理——你的语音数据永远不会离开你的设备。

什么是 VoiceFlow？

Section titled “什么是 VoiceFlow？”

VoiceFlow 是一款免费的、开源的 Windows 10/11 语音听写应用，利用 OpenAI 的 Whisper 模型进行语音转文字转换。与基于云端的替代方案不同，VoiceFlow 在你的计算机上 100% 本地运行，确保完全隐私并消除持续成本。

  • 自定义快捷键：使用保持或切换录音模式配置快捷键
  • 本地历史记录：SQLite 数据库本地存储可搜索的转录内容
  • 自动粘贴：文本直接出现在光标处——无需复制粘贴
  • GPU 支持：最近的更新增加了 GPU 加速以实现更快推理

  • 开源：GitHub 上的每一行代码都可审计
  • 无遥测：零跟踪、零分析、零云调用
  • 仅 RAM 处理：麦克风数据在处理期间从不接触磁盘

VoiceFlow 如何工作

Section titled “VoiceFlow 如何工作”

该应用以优雅的简洁性运行：

  1. 就绪：VoiceFlow 在系统托盘中静默等待，存在感极小
  2. 监听：使用快捷键激活并自然说话。音频仅保留在 RAM 中，具有实时幅度可视化
  3. 转录和粘贴：释放快捷键。本地 AI 瞬间处理你的音频，然后自动将文本粘贴到光标处

    • 安装浏览器扩展以保存需要认证的网页
    • 通过 OAuth 链接 Slack、Notion、GitHub
    • 直接上传文件（支持 50+ 种格式）

  3. 开始研究 用自然语言提问。SurfSense 将：

    • 搜索您的个人知识库
    • 查询已连接的外部源
    • 综合信息并提供引用
    • 生成可分享的播客摘要

  4. 团队协作 邀请团队成员并分配适当的角色。在保持访问控制的同时共享研究集合。

架构深入解析

Section titled “架构深入解析”

后端技术栈

  • FastAPI 提供高性能异步 API
  • PostgreSQL + pgvector 实现混合搜索
  • Celery + Redis 处理后台任务
  • LangGraph + LangChain 进行代理编排

前端技术栈

  • Next.js + React + TypeScript
  • Tailwind CSS + Shadcn 实现现代 UI
  • WebSocket 实现实时更新

部署方案

  • Docker + Docker Compose 容器化
  • 包含 pgAdmin 进行数据库管理
  • 支持水平扩展

进阶技巧

Section titled “进阶技巧”
  • 浏览器扩展威力：跨浏览器扩展可以保存需要认证的页面（登录后的内容），这是普通爬虫无法访问的。
  • 播客工作流：使用播客生成功能制作每日研究摘要——通勤时收听最佳。
  • 模型热切换：在对话中途切换 GPT-4、Claude 或本地 Llama，不会丢失上下文。
  • Ollama 集成：为了最大程度的隐私，使用 Ollama 在本地运行一切——数据零外泄。
  • 团队空间：为不同项目或团队创建独立的知识库。

Section titled “总结”

SurfSense 填补了 AI 研究领域的独特空白：企业级能力 + 个人控制权。
