Claude Code Development Tool: `claude-trace` Complete Recording Tool for Transparent Debugging

Jul 12, 2025
Vibe Sparking

When using Claude Code to write code, have you ever wondered:

  • How does Claude understand your instructions?
  • What exactly is written in the system prompt?
  • What tools did Claude use? What raw data was returned?
  • What happens behind the API? How many tokens did the model use?

Now, with the open-source tool claude-trace, you can see all these “behind-the-scenes” details!

📦 Installation

Section titled “📦 Installation”

Install with just one command:

🚀 One-Click Start Logging

Section titled “🚀 One-Click Start Logging”

Start Claude Code and record the entire process:

Supports more options:

FeatureCommand
Start Claude + loggingclaude-trace
Include all requests (default only logs valid conversations)claude-trace --include-all-requests
Run Claude with specific parametersclaude-trace --run-with chat --model sonnet-3.5
Extract OAuth tokenclaude-trace --extract-token
Generate HTML session reportclaude-trace --generate-html logs.jsonl report.html
Auto-generate index + summaryclaude-trace --index

🔍 What Can You See?

Section titled “🔍 What Can You See?”

Using claude-trace, you can see in the generated HTML page:

  • ✅ Claude’s system prompt (hidden instructions)
  • ✅ All raw API request and response data
  • ✅ Tool definitions and outputs (like search/file reading)
  • ✅ Claude’s “thinking process” blocks
  • ✅ Token usage statistics (including cache hit situations)
  • ✅ Interactive HTML view + search index
  • ✅ JSONL raw logs (saved in .claude-trace/ folder)

Each run generates a self-contained HTML file, viewable locally without a server, perfect for team reviews, debugging, and archiving.

📊 Session Summary & Index Features (AI-Enhanced)

Section titled “📊 Session Summary & Index Features (AI-Enhanced)”

Through the claude-trace --index command, you can also auto-generate:

  • AI-generated summary titles and overviews for all logs
  • Indexed index.html file
  • Clickable historical session list

Manage all Claude conversation records like managing Notion!

Section titled “🧑‍💻 Who Is This For?”
