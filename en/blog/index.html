Build an Efficient, Lightweight, and Controlled Spec‑Driven Workflow with OpenSpec + Claude Code

Oct 17, 2025
Vibe Sparking

When teams start collaborating with AI coding assistants, clearly answering “what to build, how to build it, and how we know it’s done” becomes crucial. This guide walks you through a practical, reviewable, and traceable development workflow using OpenSpec (documentation‑driven development) together with Claude Code (an intelligent coding assistant).

Who this is for:

  • Teams that want specs before changes in multi‑person collaboration
  • People/teams that want AI assistants embedded into a standard workflow
  • Projects that require approval, archiving, or compliance control

Core Principles — Specs Are Truth

Section titled “Core Principles — Specs Are Truth”
  • Specs are truth: openspec/specs represents the currently delivered source of truth.
  • Propose before changes: any change impacting behavior/architecture/security/performance starts in changes/[id].
  • Three‑phase loop: Create change → Implement tasks → Archive and codify.
  • AI works to specs: Claude Code always uses OpenSpec as the authoritative context.

Install and Initialize

Initialization will:

  • Write a cross‑assistant AGENTS guide at the repo root
  • Generate the openspec/ directory structure
  • Establish unified collaboration instructions for Claude Code / Cursor / Codex, etc.
  • Later you can run openspec update to refresh the guidelines

Collaborating with Claude Code

Section titled “Collaborating with Claude Code”

OpenSpec provides project context and process standards; Claude Code executes the Read → Generate → Implement → Verify cycle. After initialization, start your Claude Code session with prompts like these:

  • Fill in project context:

Please read openspec/project.md. Based on the repo’s current state, help me fully document project goals, tech stack, conventions, external dependencies, constraints, and naming standards.

  • Create the first change proposal:

I want to add “[your feature]”. Please create an OpenSpec change proposal (changes/[id]/) including proposal.md, tasks.md, and the relevant specs delta under changes/[id]/specs/.

  • Explain/enforce the workflow:

Using openspec/AGENTS.md, explain the three‑phase workflow and strictly follow it in subsequent steps: propose and validate first, implement after approval, archive when complete.

Tip: After installation, Codex/Windsurf/Cursor, etc. will also pick up the same guidelines so multiple assistants collaborate consistently.

Three‑Phase Workflow (from openspec/AGENTS.md)

Section titled “Three‑Phase Workflow (from openspec/AGENTS.md)”
  1. Create Change
  • When: new capability, breaking change, architecture/security/performance strategy changes
  • Steps:
    • Pick a unique change-id (kebab‑case, verb‑prefixed: add-, update-, remove-, refactor-)
    • Generate proposal.md, tasks.md, and design.md if needed
    • Write specs delta using operation headers: ## ADDED|MODIFIED|REMOVED|RENAMED Requirements
    • Validate: openspec validate <id> --strict
    • Only implement after approval
  1. Implement Change
  • Steps:
    • Read in order: proposal.md → design.md (optional) → tasks.md
    • Implement strictly in tasks.md order; check off each item when complete
    • Update overall task status only when all are completed
  1. Archive Change
  • After deployment:
    • Move changes/[id] to changes/archive/YYYY-MM-DD-[id]/
    • If capability behavior changed, update openspec/specs/
    • Run strict validation: openspec validate --strict
    • Optional: openspec archive [change] [--yes|-y] to auto‑archive (tooling changes may use --skip-specs)

Key Spec File Format (Important)

Section titled “Key Spec File Format (Important)”
  • Each requirement uses ### Requirement: ...
  • Each requirement must have at least one scenario using a level‑4 heading:

Do not use bullets or bold text instead of a scenario heading (the parser will fail).

  • Three types of delta changes:
    • ## ADDED Requirements for new capabilities
    • ## MODIFIED Requirements for behavior changes (paste the full updated requirement block, not a diff)
    • ## REMOVED Requirements for removals (include rationale and migration)
    • ## RENAMED Requirements for renames only

From Zero to One: A Happy Path

Section titled “From Zero to One: A Happy Path”
  1. Explore current state
  1. Choose a change ID and scaffold
  1. Write the delta (example)
  1. Strict validation
  1. Approval → Implementation
  • After approval, have Claude Code implement items in tasks.md step by step
  • Check off one item when it’s done; proceed until all are - [x]
  1. Archive and codify

Best Practices with Claude Code

Section titled “Best Practices with Claude Code”
  • Always let Claude Code “read” first: openspec/project.md, relevant specs/[capability]/spec.md, and changes/[id]/proposal.md|tasks.md
  • Execute strictly against tasks.md as the single source of truth; check off as you go
  • Prioritize edits directly related to the proposed change; avoid unrelated refactors
  • When ambiguous, ask 1–2 clarifying questions before scaffolding
  • When referencing code locations, use src/module.ts:42 single‑line pointers
  • Built‑in validation: openspec validate [change] --strict; if issues occur, use --json to diagnose

CLI Cheat Sheet

Section titled “CLI Cheat Sheet”

List / show / diff / validate

Section titled “List / show / diff / validate”

Common flags:

  • --json machine‑readable output
  • --type change|spec to specify the object type
  • --strict strict validation
  • --skip-specs archive only (skip specs update)
  • --yes|-y skip confirmation (good for CI)

Proposal vs. Direct Fix — When to Use Which?

Section titled “Proposal vs. Direct Fix — When to Use Which?”

Propose (recommended) when:

  • New features/capabilities
  • Breaking changes (API/Schema)
  • Architecture/pattern shifts
  • Performance/security strategy changes that affect behavior

Direct edits are fine when:

  • Bug fixes (restoring intended behavior)
  • Copy/format/comments
  • Non‑breaking dependency upgrades
  • Minor configuration tweaks
  • Tests that cover existing behavior

Common Pitfalls and Troubleshooting

Section titled “Common Pitfalls and Troubleshooting”
  • “Change must have at least one delta”
    • Check that changes/[id]/specs/ contains .md files
    • Ensure files include ## ADDED|MODIFIED|REMOVED operation headers
  • “Requirement must have at least one scenario”
    • Scenario headings must be #### Scenario: ... (level‑4 heading)
    • Do not use bullets or bold text
  • Delta parsing failures
    • Use openspec show [change] --json --deltas-only to debug
    • Compare with originals under openspec/specs/ and ensure MODIFIED is a full replacement block

Conclusion

Section titled “Conclusion”

Use OpenSpec to make Requirements → Design → Tasks → Changes explicit from day one and treat them as the project’s source of truth. Then leverage Claude Code for efficient execution and collaboration so AI delivers within clear boundaries. The benefits:

  • Specs first to reduce rework; approvals up front to lower risk
  • Task closure with traceable outputs; archivable changes for reusable knowledge

Q3 2025 AI Chip Supply Chain Earnings Start Strong: ASML and TSMC Beat — Where Does NVIDIA Go from Here?

Oct 16, 2025
Vibe Sparking

Opening: Tech Earnings Season Kicks Off, Supply‑Chain Confidence Surges

Section titled “Opening: Tech Earnings Season Kicks Off, Supply‑Chain Confidence Surges”

As this earnings season shows, the AI chip supply chain is firing on all cylinders. ASML and TSMC both delivered upside surprises in their Q3 2025 results. More importantly, both turned distinctly more optimistic in their outlooks — a sharp contrast with their cautious tone three months ago.

ASML Q3 Highlights

Section titled “ASML Q3 Highlights”
  • Revenue €7.5B (≈$8.2B), gross margin 51.6%
  • Net income €2.125B, above market expectations
  • Q4 guidance raised sharply to €9.2–9.8B; gross margin 51%–53%
  • The key shift: management explicitly said 2026 revenue will not be below 2025, reversing prior uncertainty

TSMC’s Record Q3

Section titled “TSMC’s Record Q3”
  • Revenue NT$989.92B (≈$32.2B), up 39% y/y
  • Net income NT$452.3B (≈$14.8B), a record high
  • ADR EPS up 39% y/y; dynamic P/E only 33x, PEG < 1
  • Full‑year revenue growth target raised from 30% to 35%
  • Gross margin at a strong 59.5%, underscoring pricing power

The core message from both prints: AI acceleration is itself being accelerated.

Supply‑Chain Triangle: ASML → TSMC → NVIDIA

Section titled “Supply‑Chain Triangle: ASML → TSMC → NVIDIA”

To understand how these three companies interrelate, start with the key nodes in the AI chip value chain:

ASML — The Photolithography Monopolist

Section titled “ASML — The Photolithography Monopolist”
  • World’s only supplier of EUV lithography equipment
  • Each EUV system sells for >$200M; the latest High‑NA EUV is ~$380M per unit
  • All advanced‑node manufacturers (TSMC, Samsung, Intel) depend heavily on ASML

TSMC — The King of Advanced Foundry

Section titled “TSMC — The King of Advanced Foundry”
  • 61% global foundry share; 67% share in advanced nodes (<7nm)
  • Chips from all top AI designers (NVIDIA, Apple, AMD, etc.) are manufactured by TSMC
  • Mass‑production at 3nm; 2nm in progress

NVIDIA — The AI Chip Design Leader

Section titled “NVIDIA — The AI Chip Design Leader”
  • ~$4.38T market cap; data‑center business is ~88% of revenue
  • Relies entirely on TSMC’s advanced nodes to produce H100, H200, and the new Blackwell series
  • Blackwell generated ~$11B in FY25 Q4 revenue alone — the “fastest product ramp” in company history

The transmission goes: ASML’s EUV tools → TSMC’s advanced‑node capacity → NVIDIA’s AI chip supply. A bottleneck at any node constrains the entire chain.

Data: Stock‑Performance and Results Correlations

Section titled “Data: Stock‑Performance and Results Correlations”

Analyzing post‑earnings stock moves for these three from 2022–2025 reveals important patterns:

Finding 1: TSMC’s Link to NVIDIA Is Tighter

Section titled “Finding 1: TSMC’s Link to NVIDIA Is Tighter”

Our statistics show:

  • In the 10 trading days after TSMC earnings, NVIDIA’s return correlation is 0.70 (p=0.004) — statistically significant
  • After ASML earnings, NVIDIA’s 10‑day correlation is 0.49 (p=0.065) — weaker
  • Over 30‑day windows, TSMC↔NVIDIA correlation stays ~0.48; ASML drops to ~0.13

This supports the intuition that the nearer the link in the chain, the stronger the co‑movement. As NVIDIA’s direct supplier, TSMC’s capacity and process progress directly shape how many chips NVIDIA can ship.

Finding 2: Supplier Earnings Tend to Lift NVIDIA

Section titled “Finding 2: Supplier Earnings Tend to Lift NVIDIA”

On average:

  • In the 10 days after ASML reports, NVIDIA rose 3.8%; over 30 days, +14.2%
  • In the 10 days after TSMC reports, NVIDIA rose 2.1%; over 30 days, +15.8%
  • The effect was most pronounced in 2023 (the breakout year for AI): post‑ASML 30‑day average +19.8%; post‑TSMC +26.7%

Finding 3: Correlations Weakened in 2024–2025 but Remain Meaningful

Section titled “Finding 3: Correlations Weakened in 2024–2025 but Remain Meaningful”

As NVIDIA’s market cap surpassed $3T, its stock began to reflect more macro factors:

  • Fed policy expectations
  • U.S. export‑control policy toward China
  • Competition (AMD; hyperscaler in‑house chips)

Even so, supplier earnings remain pivotal indicators for NVIDIA — especially TSMC’s expansion plans and ASML’s EUV shipment cadence, which map to NVIDIA’s potential shipments over the next 12–18 months.

Share‑Price Performance Comparison

Section titled “Share‑Price Performance Comparison”

NVIDIA vs. ASML vs. TSMC since early 2022 (indexed to 100 at Jan 2022)

The chart shows NVIDIA up 497%, TSMC up 137%, and ASML up 27% over the period. The gap reflects differences in value capture:

  • NVIDIA: commands the highest value‑added layer via CUDA software ecosystem and leading chip design
  • TSMC: manufacturing profits are high (≈60% gross margin) but massive capex tempers stock elasticity
  • ASML: EUV monopoly, but the customer base is limited (principally TSMC, Samsung, Intel) and deliveries are long‑cycle, making growth steadier

Three Key Signals from This Earnings Season

Section titled “Three Key Signals from This Earnings Season”

Signal 1: ASML’s 2026 Stance Flips 180 Degrees

Section titled “Signal 1: ASML’s 2026 Stance Flips 180 Degrees”

In July 2024 (Q2), ASML cautioned that “given rising macro and geopolitical uncertainty, we cannot confirm growth in 2026,” triggering a ~16% single‑day drop.

In Q3, management stated clearly that “2026 total revenue will not be below 2025,” stressing enduring AI‑driven demand. Drivers include:

  • Strong EUV orders: €5.4B net bookings in Q3, with €3.6B from EUV
  • High‑NA EUV shipments have begun, at higher price points (~$380M/unit)
  • TSMC, Samsung, Intel are all expanding; photolithography demand remains robust

Signal 2: TSMC Lifts 2025 Full‑Year Growth from 30% to 35%

Section titled “Signal 2: TSMC Lifts 2025 Full‑Year Growth from 30% to 35%”

In Q3, TSMC raised its 2025 USD revenue‑growth outlook from 30% to 35% — the second raise this year. CFO Wendell Huang noted: “We saw strong demand for advanced technologies in Q3, and as we enter Q4, we expect the strength to continue.”

More importantly, TSMC said 3nm and more advanced nodes are fully booked, with customers “competing for capacity,” indicating strong demand from NVIDIA, Apple, and others.

Signal 3: NVIDIA Blackwell Is Supply‑Constrained; Q4 Guide Above Street

Section titled “Signal 3: NVIDIA Blackwell Is Supply‑Constrained; Q4 Guide Above Street”

While NVIDIA has not yet reported Q3 (due Nov 19), Q2 already showed a rapid Blackwell ramp:

  • Q2 Blackwell revenue +17% q/q
  • Q3 guide $54.0B (±2%), above the Street’s $52.7B
  • CEO Jensen Huang: “Blackwell demand is surprising… we have achieved large‑scale mass production of Blackwell AI supercomputers.”

UBS lifted its Blackwell Q4 revenue estimate from $5B to $9B, signaling demand far above prior expectations.

  • Q4 guide: $60–65B as Blackwell continues to ramp

Global EV sales keep setting records — >20M expected in 2025, roughly a quarter of auto sales — driving exponential battery demand. Grid‑scale storage is surging to firm intermittent wind and solar. Today’s industry is led by China, Korea, and Japan, with China dominant in capacity and key processes. The U.S., recognizing vulnerability in this supply chain, elevated batteries to a strategic priority: the 2022 Inflation Reduction Act (IRA) offers rich tax credits, catalyzing U.S. manufacturing across cells and materials. As a result, announced U.S. capacity could reach ~1.2 TWh by 2030, up ~70% versus pre‑IRA plans.

Supply–demand and outlook. Bain estimates global battery demand will climb from ~1,000 GWh in 2023 to ~4,100 GWh by 2030 (4× growth). Beyond passenger EVs, commercial vehicles, tools, and grid storage add to demand. IEA pegs current EV+storage battery market value at ~$120B, potentially near $500B by 2030. U.S. demand is also accelerating: EV leaders (e.g., Tesla) lift shipments while data centers emerge as a new storage buyer. Nearly 95% of incremental power demand growth is expected to be met by PV and wind, which in turn relies on large‑scale storage. Fastmarkets notes data‑center power usage has grown ~12% annually since 2017; data centers are installing substantial battery backup and could account for roughly one‑third of storage demand by 2030. On the supply side, IRA’s $35/kWh production credit draws foreign JV investment and domestic scale‑ups. Short‑term challenges persist: rapid tech iteration, heavy capex, and 2023’s higher rates/policy uncertainty led to delays or cancellations for some plants. Medium term, demand growth looks robust, but localized gluts and uneven utilization could occur — watch for over‑build and intensifying competition.

Representative companies & investment take. The segment spans vehicles, cells, systems, and materials: Tesla (TSLA), Eos Energy (EOSE), Microvast (MVST), American Battery Technology (ABAT), among others. Valuation, however, already embeds high growth. Eos targets zinc‑based storage for grid‑scale safety/longevity, has DOE loan support, and offers high upside with high execution risk. Broadly, policy tailwinds are clear: IRA credits can cut manufacturing cost by >30%; content rules in purchase incentives steer localized supply. Longer term, scale leaders and firms with secured orders should compound best. Beware hype cycles: some storage names rallied on narrative ahead of commercialization — differentiate mature, profitable materials leaders from early‑stage stories; balance “core” stable names with a small “satellite” allocation to higher‑beta growth, per risk tolerance.

Rare Earths & Strategic Metals: Industry Vitamins and Defense Arteries

Section titled “Rare Earths & Strategic Metals: Industry Vitamins and Defense Arteries”

Industry context and strategic role. “Rare earths” (17 elements) are often called the vitamins of modern industry — essential for high‑performance permanent magnets (EV motors, wind turbines), avionics, guidance systems, and semiconductors. Strategic metals (e.g., antimony, cobalt, titanium, tungsten) are key to defense/high‑tech. The U.S. Department of Defense (DoD) has repeatedly warned that rare earths are indispensable to advanced weapon systems (F‑35s, submarines, missiles, UAVs). In practice, the U.S. relies heavily on China: China mines ≥60% and processes ~90% of rare‑earth products; 70% of U.S. rare‑earth compounds/metals imports (2020–2023) came from China. China is also a major supplier of antimony (armor alloys, fuzes), graphite (battery anodes), and tungsten (AP rounds, tooling). This dependency is viewed as a national‑security weak point. In 2025, China tightened controls further on rare earths and magnet exports (declaring uses for products containing heavy RE), heightening global concern. Analysts see it as signaling that Beijing may wield rare‑earth supply as leverage. The U.S. response includes reducing reliance, tariff tools, and larger resource investment.

Supply–demand and outlook. Rare‑earth demand has surged with EVs and clean energy, while supply growth remains concentrated. Export curbs by key suppliers could cause shortages and price swings. The U.S. is acting on multiple fronts: DoD funding to MP Materials for separation, and to Lynas for heavy‑RE processing in Texas; Defense Production Act (DPA) support for lithium, graphite, nickel, antimony, etc. After China restricted antimony exports in Aug‑2024, the Defense Logistics Agency (DLA) expanded procurement plans, reportedly earmarking ~$245M with U.S. Antimony (UAMY) — roughly three‑quarters of annual U.S. imports. Strategic stockpiles are set to expand: reports in Oct‑2025 suggested a ~$10B critical‑minerals reserve to reduce China reliance. Over 2025–2030, new projects in Australia/Canada could diversify supply; deep‑sea mining (e.g., The Metals Company targeting nickel/cobalt nodules) may contribute. But permitting, costs, and price cycles complicate investment — China’s position remains strong in the near term, with allied cooperation and state support needed to bridge gaps. Geopolitical risk will be persistent; decoupling is partial at best, implying a protracted, uneven supply‑chain re‑wiring.

Representative companies & investment take. The cohort includes MP Materials (MP), USA Rare Earth (USAR), U.S. Antimony (UAMY), and Ivanhoe Electric (IE). MP — owner of Mountain Pass, the only producing U.S. RE mine — is core to U.S. revival efforts, building separation and magnet capacity with government backing, and is among the more mature, revenue‑generating names. USA Rare Earth controls Texas’ Round Top deposit (heavy RE + lithium, etc.) and has been bid up post‑listing on strategic value; it remains pre‑revenue and high beta. UAMY is one of few North American antimony producers, small in scale but strategically notable given DLA contracts; fundamentals hinge more on policy than on structural growth. Overall, many names are early stage with elevated hype risk. Focus on who can deliver capacity and long‑term offtakes. Policy can whipsaw prices (bans, tariffs, contracts). Resource equities are also rate/commodity‑cycle sensitive; small caps are often first to sell off when the dollar strengthens or commodities fall.

Lithium: The “White Oil” of the EV Era

Section titled “Lithium: The “White Oil” of the EV Era”

Industry context and strategic role. Lithium — the “white oil” of the energy‑transition era — is indispensable to Li‑ion batteries. With EVs and storage booming, it has become a strategic resource. The U.S. classifies lithium as critical and sees it as central to EV‑supply‑chain autonomy. Domestic resource endowment is modest: only one producing site today (Silver Peak brine, NV), far short of needs. Much of U.S. lithium arrives via imports from South America/Australia and is then refined in China. USGS estimates U.S. dependence on lithium compounds exceeded 50% in 2022. Any geopolitical disruption or export control poses EV supply risks. Washington is backing U.S. projects (e.g., DOE loans/guarantees to Lithium Americas’ Thacker Pass; R&D into clay/geothermal‑brine extraction) and tying IRA consumer incentives to sourcing from the U.S. or FTA partners — elevating lithium’s strategic importance.

Supply–demand and outlook. Lithium saw a roller‑coaster: a 2021–2022 price spike on EV demand, followed by a 2023 correction as supply growth outpaced demand. Fastmarkets estimates 2024 lithium supply up ~22% (China, Australia, South America, Africa), while demand remained strong but temporarily lagged, creating a near‑term surplus. Medium to long term, the outlook remains positive: global lithium consumption in 2024 is ~220kt LCE (+29% y/y), with EV demand projected to grow 12% annually through 2030. At that pace, 2030 demand could be >3× today. New supply is coming: faster expansions in Australia/South America; Africa could contribute 18% of hard‑rock supply by 2030; U.S. projects (Thacker Pass; Arkansas geothermal brines) may come online 2026–2030. Risks: mining projects are multi‑year with tough permitting; U.S. timelines have slipped (e.g., Thacker Pass litigation). Refining is concentrated in China, where cash costs ($8k/t) are well below North America (>$20k/t), meaning ore may still need overseas processing. Closing the midstream gap requires heavy investment in U.S. chemical refining and community/ESG solutions. Tech shifts (solid‑state, sodium‑ion) could alter mix, but most forecasts still see Li‑ion dominance through 2030 — lithium’s centrality likely persists.

Representative companies & investment take. Upstream and midstream leaders include Albemarle (ALB), Lithium Americas (LAC), and Sigma Lithium (SGML). Albemarle, a top global producer with Chilean brines and Australian rock, is mature and cash‑generative, and plans U.S. refining capacity — a core, more defensive exposure, albeit earnings remain price‑sensitive. Lithium Americas is developing Thacker Pass, the largest U.S. clay resource, with GM investment and government support; 2026 start‑up is targeted, leaving the company in investment mode with valuation reflecting resource optionality — monitor progress and capex closely. Sigma operates high‑grade Brazilian assets and has begun trial output; an emerging cash‑flow name but subject to rumor‑driven swings. After prior exuberance, valuations have reset, offering long‑term entry points — but differentiate by stage: producers are more resilient; developers offer higher upside with higher risk. A prudent approach is a “core + satellite” mix: core allocation to leaders for baseline returns, plus smaller satellite bets on high‑beta names for potential alpha. Mind the price cycle — build on weakness and avoid chasing peaks.

JPMorgan CEO Jamie Dimon has said the U.S. has relied too much on unreliable supply chains for security‑critical minerals and products — a vulnerability now laid bare. Hence the government’s industrial‑policy toolkit — subsidies, tariffs, CFIUS‑style reviews, and export controls — is being deployed to reshape key sectors. Carbon‑neutral commitments also require massive clean‑energy infrastructure, turbo‑charging demand for lithium, cobalt, nickel, rare earths, etc. Meanwhile, geopolitical shocks (trade tensions, conflicts) amplify supply uncertainty; countries increasingly securitize resources (e.g., China’s repeated curbs on critical‑mineral exports; the U.S. using the NDAA and related measures to prioritize domestic supply). Second, “friend‑shoring” is real: U.S. alliances with resource‑rich partners (Australia, Canada, parts of Africa) aim to build diversified, lower‑risk networks (e.g., the Minerals Security Partnership). Expect a more distributed raw‑materials map and more cross‑border opportunities — but also local instability and competition with Chinese capital. Third, macro cycles matter: inflation can lift commodity prices and resource cash flows, but higher rates raise project financing costs and compress multiples for growth names. The recent hiking cycle pressured renewables/storage equities; over the long run, however, real‑asset scarcity can trump monetary headwinds if structural shortages emerge. Investors should weigh macro and industry cycles and time entries — adding on weakness to policy‑supported quality names can be effective.
