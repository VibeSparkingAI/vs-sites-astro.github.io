DeepTutor: AI-Powered Personalized Learning Assistant Based on Multi-Agent Architecture

Section titled “DeepTutor: AI-Powered Personalized Learning Assistant Based on Multi-Agent Architecture”

In today’s rapidly evolving landscape of AI-driven education, traditional one-size-fits-all tutoring approaches are giving way to intelligent, adaptive learning systems. Allow me to introduce DeepTutor — an open-source AI personalized learning assistant from the Data Intelligence Lab at The University of Hong Kong (HKUDS), which combines multi-agent architecture with advanced RAG capabilities to deliver a comprehensive learning experience.

What is DeepTutor?

Section titled “What is DeepTutor?”

DeepTutor is a full-stack AI learning platform that revolutionizes how students and researchers interact with educational content. Open-sourced under the AGPL-3.0 license, it offers seven interconnected modules designed to support the entire learning lifecycle, from knowledge acquisition to research ideation.

Core Advantage: DeepTutor employs a dual-loop reasoning architecture that separates the investigation process from problem solving, enabling deeper analysis and more accurate solutions compared to single-turn AI tutors.

1. Smart Solver — Dual-Loop Problem Solving

Section titled “1. Smart Solver — Dual-Loop Problem Solving”

The Smart Solver employs a sophisticated dual-loop architecture that separates analysis from solving:

Analysis Loop

  • InvestigateAgent: Collects context and relevant materials
  • NoteAgent: Compresses findings into structured notes

Solve Loop

  • PlanAgent: Creates step-by-step solution plans
  • ManagerAgent: Coordinates overall execution
  • SolveAgent: Generates solutions using tools
  • CheckAgent: Verifies answers
  • FormatAgent: Formats final output

FAQ

Section titled “FAQ”

Q: What if port 8001 is already in use? A: Edit config/main.yaml and change server.backend_port to another port.

Q: Can I add documents to an existing knowledge base? A: Yes! Use incremental addition: python -m src.knowledge.add_documents <kb_name> --docs <new_file.pdf>

Q: How do I properly stop the service? A: Press Ctrl+C. If the port remains occupied, use lsof -i :8001 to find and terminate the process.

Q: Does it support offline operation? A: After knowledge base construction, core RAG functions can run offline. Web search and paper search require internet connection.

SurfSense is an open-source AI research agent that combines personal knowledge management with powerful external integrations. Think NotebookLM + Perplexity + Glean—but fully under your control. The modern researcher’s dilemma: NotebookLM is great for documents, Perplexity excels at web search, and Glean conquers enterprise search—but what if you want all three, connected to YOUR data, running on YOUR infrastructure?

Section titled “Quick Start Workflow”

  1. Launch & Configure After installation, navigate to localhost:3000 and configure your LLM provider (OpenAI API key, or connect Ollama for local inference).

  2. Connect Your Sources

    • Install the browser extension to save authenticated web pages
    • Link Slack, Notion, GitHub through OAuth
    • Upload files directly (supports 50+ formats)

  3. Start Researching Ask questions in natural language. SurfSense will:

    • Search your personal knowledge base
    • Query connected external sources
    • Synthesize information with citations
    • Generate shareable podcast summaries
