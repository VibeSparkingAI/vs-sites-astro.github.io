Section titled вҖңдёәд»Җд№ҲиҰҒиҮӘе·ұжҗӯе»ә Agent е№іеҸ°пјҹвҖқ

Claude Code жҳҜ Anthropic зҡ„ AI зј–зЁӢеҠ©жүӢгҖӮSDK и®©дҪ еҸҜд»ҘжҠҠе®ғеөҢе…ҘиҮӘе·ұзҡ„зі»з»ҹгҖӮ

й—®йўҳжқҘдәҶпјҡеҰӮжһңжғіи®©еӨҡдёӘеӣўйҳҹе…ұз”ЁдёҖеҘ— Agent жңҚеҠЎпјҢеҗҢж—¶дҝқиҜҒж•°жҚ®йҡ”зҰ»пјҢжҖҺд№ҲеҠһпјҹ

еӨ©жһў (TianShu) жҳҜдёҖдёӘејҖжәҗж–№жЎҲгҖӮе®ғжҠҠ Claude Code дјҡиҜқи·‘еңЁйҡ”зҰ»зҡ„ Worker дёҠпјҢйҖҡиҝҮ Web UI з»ҹдёҖз®ЎзҗҶгҖӮдёӢйқўзңӢзңӢе…·дҪ“жҖҺд№Ҳе®һзҺ°зҡ„гҖӮ
